# TG03

##### РЕШЕНИЕ В ФАЙЛЕ *school_data.py*
##### КОМАНДА */showschool* ОТПРАВЛЯЕТ ПОЛЬЗОВАТЕЛЮ ЗАПИСИ БАЗЫ ДАННЫХ school_data.py

Создайте новую базу данных school_data.db. 

В этой базе данных создайте таблицу students с колонками: 
* id (INTEGER, PRIMARY KEY, AUTOINCREMENT) 
* name (TEXT) 
* age (INTEGER) 
* grade (TEXT)

Создайте Телеграм-бота, который запрашивает у пользователя 
* имя, 
* возраст
* класс (grade), в котором он учится. 

Сделайте так чтоб бот сохранял введенные данные 
в таблицу students базы данных school_data.db.
